Plugin duplicates the posts, pages all at once based on the post type,post status and even year of posts created.

== Description ==

This plugin used to quickly and easily duplicate posts, pages , custom post.By default the post with be duplicated with "draft" status.

To create duplicate, Just hover over a posts, pages , custom post in the edit screen and select 'Duplicate' to create a duplicate post.


== Installation ==
To install wp rks duplicator using the built-in plugin installer:

    1. Go to Plugins > Add New.
    2. Type in wp rks duplicator in Search Plugins box or click a tag link below the screen.
    3. find the wp rks duplicator Plugin in the list.
    4. Click Install Now to install the WordPress Plugin.


To install wp rks duplicator manually:

   1. Download your WordPress Plugin to your desktop.
   2. If downloaded as a zip archive, extract the Plugin folder to your desktop.
   3. With your FTP program, upload the Plugin folder to the wp-content/plugins folder in your WordPress directory online.
   4. Go to Plugins screen and find the wp rks duplicator Plugin in the list.
   5. Click Activate to activate it.

== Frequently Asked Questions ==

= Does this plugin support custom post types? =

Yes, all registered custom post types are supported.

= How to duplicate custom post types? =

Go to wp-admin >> pages/posts/custom post , On quick edit bar click on duplicate.
